Introduction¶

This dataset contains information about thousands of movies collected from TMDb.

Notes:

  • Certain columns, like ‘cast’and ‘genres’, contain multiple values separated by pipe (|) characters.
  • There are some odd characters in the ‘cast’ column. I am leaving them as is.
  • The final two columns ending with “_adj” show the budget and revenue of the associated movie in terms of 2010 dollars, accounting for inflation over time.

A lot of insight can be garnered from this dataset and we will go about exploring some of these insights in this report. Here are some questions that I am curious to discover the answer to:

  • Top 10 successful movies and their genres.
  • Most popular movie genres and the least popular.
  • Most successful actors.
  • Which month are movies released in the most.
  • Which month are the most profits made.

# This function which will take any column as argument from which data is to be extracted and keep track of the count
def extract_data(col):
    # This should take a column, and separate the string by '|'
    extracted = cd[col].str.cat(sep = '|')
    
    # Returning pandas series and storing the values separately
    extracted = pd.Series(extracted.split('|'))
    
    # Sorting values in descending order
    count = extracted.value_counts(ascending = False)
    
    return count
